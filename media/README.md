# Django E-commerce Project

Mahsulot, savat va buyurtmalarni boshqaruvchi Django internet-do‘koni.

## Imkoniyatlar

- Mahsulot va kategoriyalar
- Qidiruv va filter
- Mahsulot rasmlari
- Session asosidagi savat
- Register, login va logout
- Buyurtma rasmiylashtirish
- Ombordagi stock nazorati
- Buyurtmalar tarixi
- Admin panel
- Avtomatik testlar
